### 2. Architecture Design (project-architect agent)
Designed the complete MVP blueprint:

**Database — 7 tables:**
- `users` — customers, staff, and managers in one table (role-based)
- `locations` — rental branch locations
- `vehicles` — car and motorcycle inventory
- `bookings` — core rental transaction table with status state machine
- `booking_addons` — optional add-ons per booking (insurance, GPS, etc.)
- `payments` — one payment record per booking
- `reviews` — customer reviews on completed bookings

**REST API — 35+ endpoints across 7 route groups:**
`/auth`, `/vehicles`, `/locations`, `/bookings`, `/payments`, `/reviews`, `/reports`

**MVP Scope defined:**
- Customer browse/book/cancel flow
- Staff confirm/pickup/return workflow
- Manual payment recording (no payment gateway)
- Manager revenue and usage reports

**Out of scope for now:** Stripe/PayPal, image uploads, email notifications, discount codes

### 3. Backend (backend-agent)
Built the complete Python/FastAPI backend — 32 files in `backend/`:

**Key files:**
- `main.py` — FastAPI app entry point, CORS, all routers
- `config.py` — environment variable management
- `database.py` — SQLAlchemy engine and session
- `models/` — 6 ORM model files (User, Vehicle, Location, Booking, Payment, Review)
- `schemas/` — Pydantic request/response schemas for all models
- `services/auth_service.py` — JWT creation/decoding, password hashing, role guards
- `services/booking_service.py` — availability check (SELECT FOR UPDATE), price calculation, cancellation fee logic
- `routes/` — 7 route files implementing all API endpoints
- `database/seed.py` — seeds 3 locations, 10 vehicles, 1 manager, 2 staff, 5 customers
- `requirements.txt`, `.env.example`

**Critical business logic implemented:**
- Double-booking prevention via `SELECT ... FOR UPDATE` inside a transaction
- Booking state machine: `pending → confirmed → active → completed` (with `cancelled` as exit at any non-terminal state)
- Cancellation fees: 0% (>7 days), 25% (3–7 days), 50% (<3 days), 100% (same day)
- Addon pricing: Insurance ₱15/day, GPS ₱5/day, Extra Driver ₱10/day, Child Seat ₱8/day

**Bug fixed during setup:**
- Removed `unsigned=True` from all SQLAlchemy `mapped_column()` calls (not supported in SQLAlchemy 2.x)
- Replaced `passlib` with direct `bcrypt` calls (passlib incompatible with newer bcrypt versions on Python 3.13)
